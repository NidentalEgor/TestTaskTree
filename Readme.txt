Проект был создан в Visual Studio 2015.

Описание задания в файле TestTaskTree.pdf.

Структура дерева в файле:

неограниченное кол-о записей вида
<тип_узла количество_потомков значение_узла>

тип_узла:
0 - целое число
1 - вещественное число
2 - строка

СериализациЯ в консоль:
Узлы разных уровней разделены строкой "Level #".
Потомки узла в рамках одного уровнЯ разделены пробелом ' '. 
Потомки разных узлов в рамках одного уровнЯ разделены вертикальной чертой '|'.

Чтобы собрать проект нужен boost (я использовал 1.62) и
переменная среды BOOST_ROOT, указывающая на папку с boost-ом.

В проекте Tests в папке TreeTests есть тестовые данные,
на которых можно позапускать приложение с сериализацией в
консоль.
